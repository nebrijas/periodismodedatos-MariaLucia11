Creación de Dataframe¶

Es una tabla a la que llamamos Dataframe, vamos a conectarnos a la tabla de la API y vamos a traer algunos. Como esta Api devuelve los datos en JSON, utilizamos la función read_json de Pandas para leer los datos en JSON de la API:

A nuestro dataframe lo vamos a llamar *df. Nuestra tabla de datos será la lectura y la url.

In [8]:
df = pd.read_json(url)

Para comprobar que tenemos algo ahí, le decimos que muestre el dataframe y vemos una tabla de datos con 248 filas y 3 columnas. Lo que sale es un extracto y aparecen solo tres columnas porque pandas tiene una columna a la que la llama índice y no la considera como una columna de datos. Los datos que hemos extraido de la url tiene una columna que se llama país, slug y *el código internacional de cómo se llaman los paises.
